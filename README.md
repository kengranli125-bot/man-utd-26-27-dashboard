# UNITED 26/27

曼联 2026/27 赛季网页看板，包含英超、欧冠、足总杯、联赛杯、季前赛、赛果、英超与欧冠积分榜、球队射手榜、球队动态、一线队状态评估及英超/欧冠预测首发。

主页面使用曼联官网 `Visit Old Trafford` 页面加载的老特拉福德实景图。球员资料以 ESPN 为主，FotMob 公开阵容页面用于补充真人头像；球员卡展示中文姓名、号码、位置、年龄、国籍与公开状态。缺少可靠照片的球员会显示姓名缩写占位，不生成虚构肖像。

## 本地查看

```powershell
python -m http.server 4173
```

打开 `http://localhost:4173`。页面必须通过 HTTP 服务访问，因为数据使用 `fetch` 读取。

## 每日更新

GitHub Actions 每 6 小时运行一次 `scripts/update-data.mjs`（UTC 00:15、06:15、12:15、18:15；北京时间 08:15、14:15、20:15、02:15），从 ESPN 公开接口聚合英超、欧冠、足总杯、联赛杯和俱乐部友谊赛，同时拉取新闻与一线队名单，通过 MyMemory 免密钥接口生成中文新闻标题，并提交 `data/dashboard.json`。尚未抽签的杯赛会在赛程发布后自动加入。阵容模型会根据公开状态和近期伤病新闻更新可用性评估，并自动避开伤缺球员生成英超与欧冠预测首发。也可以在 Actions 页手动运行 `Six-hour dashboard update`。

已打开的页面每 10 分钟检查一次新数据，切回浏览器标签页时也会立即检查。`scripts/check-update-health.mjs` 用于验证数据更新时间没有超过预期窗口。

主页面的欧冠联赛阶段积分榜和曼联射手榜使用 26/27 赛季接口，不回填上赛季数据。欧冠联赛阶段或英超尚未产生统计时显示等待状态，数据产生后由每日任务自动填充。

更新脚本会从新闻中识别曼联转会动态并区分“已完成 / 推进中 / 租借”。已确认但 ESPN 阵容接口尚未同步的新援会通过补充目录合并进一线队名单，并在数据源正式收录后按英文姓名自动去重。

26/27 赛程尚未进入数据源时，页面会显示明确的等待状态，不会生成虚构比赛。接口临时失败时脚本保留上次成功数据。
