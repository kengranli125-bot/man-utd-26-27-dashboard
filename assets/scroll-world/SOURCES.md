# Heritage image sources

- `stadium.webp`: Old Trafford photograph loaded by Manchester United's public `Visit Old Trafford` page; existing project asset.
- `trophies.jpg`: [Manchester United trophy cabinet](https://commons.wikimedia.org/wiki/File:Manchester_United_trophy_cabinet.jpg), Wikimedia Commons.
- `kits.jpg`: [Shirt Worn by Ryan Giggs (Manchester United Museum) (262772785)](https://commons.wikimedia.org/wiki/File:Shirt_Worn_by_Ryan_Giggs_(Manchester_United_Museum)_(262772785).jpg), Wikimedia Commons.
- `crest.png`: Manchester United team mark served by ESPN's public team-logo CDN.
- `player-*.png`: Current Manchester United player headshots served by FotMob's public player-image CDN.

The generated `.webp` files are local presentation crops/composites of these sources. Refer to each linked Commons file page for its author and license version.
